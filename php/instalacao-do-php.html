O que eu preciso?

Para começar a usar o PHP, você pode:

  • Encontre um host com suporte a PHP e MySQL
  • Instale um servidor web em seu próprio PC e, em seguida, instale PHP e MySQL

Use um host da Web com suporte PHP

Se o seu servidor ativou o suporte para PHP, você não precisa fazer nada.

Basta criar alguns .php arquivos, colocá-los em seu diretório web e o servidor irá analisá-los automaticamente para você.

Você não precisa compilar nada ou instalar nenhuma ferramenta extra.

Como o PHP é gratuito, a maioria dos hosts da web oferece suporte a PHP.

Configure o PHP em seu próprio PC

No entanto, se o seu servidor não suportar PHP, você deve:

  • instalar um servidor web
  • instalar PHP
  • instalar um banco de dados, como MySQL

O site oficial do PHP (PHP.net) tem instruções de instalação para PHP: http://php.net/manual/en/install.php

Compilador/Editor PHP Online

Com o compilador PHP online da w3schools, você pode editar o código PHP e visualizar o resultado em seu navegador.
