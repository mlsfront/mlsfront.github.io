Uma string é uma sequência de caracteres, como "Hello world!".

Funções de String PHP

Neste capítulo, veremos algumas funções comumente usadas para manipular strings.

strlen() - Retorna o comprimento de uma string

A função PHP strlen() retorna o comprimento de uma string.

Exemplo

Retorna o comprimento da string "Hello world!":

<?php
echo strlen("Hello world!"); // outputs 12
?>

str_word_count() - Contar palavras em uma string

A função PHP str_word_count() conta o número de palavras em uma string.

Exemplo

Conte o número de palavras na string "Hello world!":

<?php
echo str_word_count("Hello world!"); // outputs 2
?>

strrev() - Inverte uma String

A função PHP strrev() inverte uma string.

Exemplo

Inverta a string "Hello world!":

<?php
echo strrev("Hello world!"); // outputs !dlrow olleH
?>

strpos() - Procura um texto dentro de uma string

A função PHP strpos() procura um texto específico dentro de uma string. Se uma correspondência for encontrada, a função retornará a posição do caractere da primeira correspondência. Se nenhuma correspondência for encontrada, retornará FALSE.

Exemplo

Procure o texto "world" na string "Hello world!":

<?php
echo strpos("Hello world!", "world"); // outputs 6?>

Dica: A primeira posição do caractere em uma string é 0 (não 1).

str_replace() - Substitui texto dentro de uma string

A função PHP str_replace() substitui alguns caracteres por outros caracteres em uma string.

Exemplo

Substitua o texto "mundo" por "Dolly":

<?php
echo str_replace("world", "Dolly", "Hello world!"); // outputs Hello Dolly!
?>
