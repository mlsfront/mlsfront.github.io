Um script PHP é executado no servidor e o resultado HTML simples é enviado de volta ao navegador.

Sintaxe básica do PHP

Um script PHP pode ser colocado em qualquer lugar do documento.

Um script PHP começa <?php e termina com ?>:

<?php
// PHP código vai aqui
?>

A extensão de arquivo padrão para arquivos PHP é " .php".

Um arquivo PHP normalmente contém tags HTML e algum código de script PHP.

Abaixo, temos um exemplo de um arquivo PHP simples, com um script PHP que usa uma função PHP embutida " echo" para gerar o texto "Hello World!" em uma página da web:

Exemplo

<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>

Veja o resultado:

Nota: as instruções PHP terminam com um ponto e vírgula ( ;).

Sensibilidade de maiúsculas e minúsculas do PHP

Em PHP, palavras-chave (por exemplo if, else, while, echo, etc.), classes, funções e funções definidas pelo usuário não diferenciam maiúsculas de minúsculas.

No exemplo abaixo, todas as três instruções de eco abaixo são iguais e válidas:

Exemplo

<!DOCTYPE html>
<html>
<body>

<?php
ECHO "Hello World!<br>";
echo "Hello World!<br>";
EcHo "Hello World!<br>";

?>

</body>
</html>

Veja o resultado:

Nota: No entanto; todos os nomes de variáveis diferenciam maiúsculas de minúsculas!

Veja o exemplo abaixo; apenas a primeira instrução exibirá o valor da $color variável! Isso ocorre porque $color, $COLOR, e $coLOR são tratados como três variáveis diferentes:

Exemplo

<!DOCTYPE html>
<html>
<body>

<?php
$color = "red";
echo "My car is " . $color . "<br>";
echo "My house is " . $COLOR . "<br>";
echo "My boat is " . $coLOR . "<br>";
?>

</body>
</html>
