Comentários CSS

Os comentários são usados para explicar o código e podem ajudar quando você editar o código-fonte posteriormente.

Os comentários são ignorados pelos navegadores.

Um comentário CSS é colocado dentro do <style> elemento e começa /* e termina com */:

Exemplo

/* This is a single-line comment */

p {
color: red;
}

Você pode adicionar comentários onde quiser no código:

Exemplo

p {
 color: red; /* Set text color to red */
}

Os comentários também podem abranger várias linhas:

Exemplo

/* This is
a multi-line
comment */
p {
color: red;
}

Comentários HTML e CSS

Com o tutorial HTML, você aprendeu que pode adicionar comentários à sua origem HTML usando a <!-- ... --> sintaxe.

No exemplo a seguir, usamos uma combinação de comentários HTML e CSS:

Exemplo

<!DOCTYPE html>
<html>
<head>
 <style>
  p {
   color: red; /* Set text color to red */
  }
 </style>
</head>
<body>

 <h2>My Heading</h2>

 <!-- These paragraphs will be red -->
 <p>Hello World!</p>
 <p>This paragraph is styled with CSS.</p>
 <p>CSS comments are not shown in the output.</p>

</body>
</html>
